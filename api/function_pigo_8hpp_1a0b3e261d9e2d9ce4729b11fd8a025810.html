inline void pigo::detail::read_wgt_(size_t &coord_pos, WS &ws, FileReader &r)

Read or count the weight value from a FileReader.

Template Parameters

  • wgt – if true, will read or count appropriately

  • W – the weight type

  • WS – the weight storage type

  • counting – if true, will count only

Parameters

  • coord_pos – the current coordinate to insert into

  • ws – the weight storage to use

  • r – the FileReader that is being read from
